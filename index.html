中文金融 Agent Skill

把投研问题拆成可复现的流程

它会把“这只票强不强”“策略靠谱吗”“财报有没有改善”翻译成数据源、策略规则、回测指标和风险清单。少一点玄学，多一点证据；少一点拍脑袋，多一点可复盘。

这个 Skill 到底管什么

它像一个有条理的金融任务调度台：先识别意图，再找数据，再建规则，最后交付能检查的报告。

投研分析

处理行情强弱、财务质量、估值变化、公告新闻和行业催化，把结论绑在数据范围和来源上。

策略回测

把自然语言策略写成明确规则，选择向量化或事件驱动回测，输出指标和基准对比。

风险复盘

检查未来函数、过拟合、停牌、涨跌停、交易成本、流动性和样本外稳定性。

工作流不绕弯

金融分析最怕一上来就给结论。这个 Skill 的节奏是先把问题钉牢，再让数据和假设一起上场。

识别意图

投研、筛选、策略、回测、优化、导出，先分清楚是哪类活。

选择数据

A 股优先 Tushare、AKShare、问财式资讯，必要时读取本地 CSV。

明确规则

标的池、信号、买卖、仓位、调仓、手续费、滑点、基准全部写清楚。

报告交付

给结论，也给证据；给收益，也给回撤；给结果，也给限制。

可以这样召唤它

这些句子适合作为作品集演示，也适合真实测试这个 Skill 的任务拆解能力。

用 $finance-trading-skill 看一下宁德时代最近三个月强不强，并说明数据来源。
用 $finance-trading-skill 回测 300750.SZ 从 2024 到现在的 20/60 日双均线策略。
用 $finance-trading-skill 设计一个高 ROE + 动量的 A 股筛选流程，重点检查过拟合和流动性。
用 $finance-trading-skill 把这个策略做成报告：收益、回撤、Sharpe、交易次数、风险提示都要有。

它不装神秘

市场不会因为一个漂亮页面就变得温柔。这个 Skill 的底线是：不自动实盘、不承诺收益、不把历史回测包装成未来预言。

它擅长把研究流程变清楚，把证据链变完整，把风险放到台面上。该兴奋的时候兴奋，该踩刹车的时候踩刹车。
